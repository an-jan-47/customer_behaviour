## 🚀 Overview

This project presents a **Customer Behaviour Dashboard** that helps analyze:

- Customer demographics  
- Subscription trends  
- Revenue distribution  
- Sales performance across categories and age groups  

It is designed for **data-driven decision making** and quick business insights.
